ubuntu使用技巧

1. vim退出不保存

有时候使用vim编辑系统文件的时候，由于没有权限无法保存，又无法退出，只用:q!可以不保存退出

2. 开启crontab日志

默认定时任务crontab是不开启日志的，需要修改/etc/rsyslog.d/50-default.conf并将下面一行的前面的注释#去掉（编辑的时候需要sudo权限）

然后重启rsyslog和cron服务
